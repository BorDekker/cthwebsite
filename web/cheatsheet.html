CMD

  1. cd: change directory
  2. mkdir: make directory
  3. dir: show directory
  4. git status: check for repository
  5. git init: making repository
  6. git status: check if repository is there
  7. git add: adding unadded files
  8. git commit -m "name of commit"
  9. git log: checking if the commit has happened
  10. make repository on Github
  11. git remote add origin link
  12. git branch -M (naam)
  13. git push -u origin main
  14. git push: pushing all files to Github
  15. git status: checking if u have mist anything
